# spring-boot-starter-micro-job

#### 介绍
micro-job是一款轻量级的分布式任务执行框架，内部集成了quartz框架来完成任务的分布式调度，quartz是一个强大的任务执行框架，但是quartz为我们提供的功能却是有限，我们较为关心的执行日志采集、任务失败重试、任务权重调度等，在原生的`quartz`框架内实现会较为麻烦。
